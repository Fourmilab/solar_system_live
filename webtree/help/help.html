Solar System Live Help

Help is available on the following topics. You can display help for any control in the Solar System Live display panel by clicking its label.

  • Custom settings
  • Date and time specification
  • Ephemeris field definitions
  • Heliocentric viewpoint
  • Image size
  • Colour scheme
  • Observing site latitude and longitude
  • Orbit plotting and scaling
  • Planet icons and images
  • Planets: all or inner planets only
  • Stereo viewing
  • Tracking asteroids and comets
  • Why “Orrery”?

All the controls shown in the help pages are “live”; you can press the “Update” button next to any set of controls to display an image with the given settings or, if you like, change the settings given in the help page before pressing “Update” to observe the effect. Use your browser's “Back” button to return to the help page after viewing the results.
